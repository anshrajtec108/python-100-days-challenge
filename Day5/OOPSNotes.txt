# OOP METHODS CHEAT SHEET

1. Instance Method (self)
- Works on object data
- Can access instance variables
- Can access class variables
Syntax:
    def method(self): ‚Ä¶

2. Class Method (cls)
- Works on class-level data
- Modifies class variables
- Cannot access object instance directly
Syntax:
    @classmethod
    def method(cls): ‚Ä¶

3. Static Method
- Utility/helper function inside class
- Does not use self or cls
- Related logically to class
Syntax:
    @staticmethod
    def method(): ‚Ä¶

# When to use what?

Use Instance Method when:
- You want to access/update object attributes

Use Class Method when:
- You want to access/update class attributes
- You need alternative constructors

Use Static Method when:
- You want a helper function inside class
- You don‚Äôt need class or object data
